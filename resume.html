Recent Projects

LAN Scanner

This Python script does two things. It runs an nmap scan against my home network to get the currently connected devices. It then does an ARP scan to get the devices that have been recently connected. If anyone attempts to connect to the router from a device that I know shouldn't be connected to the network normally, then it will be blacklisted. In order to connect it will have to be manually added to an allowed devices list by me that contains the MAC addresses of the devices I know are normally connected to the network.


Raspberry-Pi Home Security Camera

I had an old Raspberry Pi laying around that I was using for Netflix and XBMC. After a few months of it laying around I decided to do something with it but wasn't sure what. After seeing some out of the ordinary activity around the apartment I live in, I started researching into turning it into a home security system.

I bought a dummy CCTV camera off of Amazon, tore it apart and got to getting it to fit inside the dummy camera. Rigged up the fake red LED lights around the lens of the camera so that the camera looked more realistic. As for the programming side of things, I used OpenCV and a few other Python libraries that allowed me to build a camera that had Infrared, Night-Vision, Motion Detection, and opt-in facial recognition.


Job Scraper

One of the other recent projects that I completed is an automated web scraper that I created using Python. I wrote this program over the span of a few months when I was looking for a new job. I had been wanting a job with Ohio University for quite some time and decided I would write an automated web-scraper that would scan the pages on OU's job postings site and alert me to when they posted and positions that matched my criteria.

The script would run in a crob job and few days it would scrape the job's site and create a list of all the jobs that were matched by a predefined keywords list. When matched, it would build a dictionary of all the jobs that were found and would then email and text me the job titles and links directly to the application.
